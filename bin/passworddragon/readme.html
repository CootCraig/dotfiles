1. Overview

Did you forget the password for the account which you have created on a website? Are you trying your luck to login into the website with all the different combination of the standard password which you always use? Tired of keeping track of all the passwords for your different accounts?. It's time to organize all your userid's and passwords in one place safely.

Keep all your account information with user id,password,url and notes information in one place. With Password Dragon, you can organize passwords in multiple files. Data files will be encrypted. Each and every file has to be assigned with a password which will be asked when the file is open for viewing/editing. You can choose to store any personal data which can be stored in tabular form.

This also includes a flexible and configurable random password generator.

2. Major Features

  • Password Dragon is a freeware
  • Can run on ALL platforms as this is written in Java
  • Organize all your userid,password,url and notes information in one file
  • Store additional information using the additional attributes.
  • Data files are encrypted with BlowfishJ algorithm
  • Command line extension. Ability to view records from the command line without launching GUI.
  • Available in Multi Language.
  • Built-in random password generator
  • Copy UserId and password to clipboard
  • Password Strength indicator
  • Backup Facility
  • Application Inactive timeout. i.e Lock the aplication if not used for x minutes
  • File Locking Facility
  • Unlimited records and data files
  • Launch URL from the application
  • Either Choose browser for URL launch or use system default browser.
  • Records can be grouped using user defined categories
  • Import records from text file
  • Export all/selected records to text,HTML and XML file
  • Search records on all/specific columns
  • Sorting based on column names
  • Password can be masked in the main table
  • Ability to specify the startup sort order and startup category
  • Ability to choose the look and feel. (Kunststoff Look and Feel is the default.)
  • etc.

 

3. Installation

New Installation

Install/Verify the Java Runtime Environment :

Make sure you have java runtime environment installed in your machine. Your java version should be 1.5 or above. Type java -version in the command prompt, you should get something similar to the following.
java version "1.6.0_23"
Java(TM) SE Runtime Environment (build 1.6.0_23-b05)
Java HotSpot(TM) Server VM (build 19.0-b09, mixed mode)

If java runtime is not installed in your machine, install it from http://www.java.com and following the instruction to install it on your platform.

Install Password Dragon :

Unzip the passworddragon.zip , which will create passworddragon folder and all files will be put underneath this directory. The following files are distributed with this zip file.

passworddragon.jar This is the application package.
lib This directory contains all the libraries needed by the Password Dragon
readme.html This file
pwm.ico Icon for Password Dragon

Run Password Dragon :

Setup is done. Double click on the passworddragon.jar to bringup the application. (or) type java -jar passworddragon.jar from the install directory.

Upgrade from previous version. :

  • Take a backup of all the data files which you are using to a different location.
  • Unzip the passworddragon.zip to the same directory structure as you did in previous version.
